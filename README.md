# Sistema de Cadastro e Login com Dashboard

Este projeto é um sistema web simples com **frontend em HTML, CSS, JS** e **backend em Node.js/Express**, utilizando **MySQL** para o banco de dados.  
O sistema permite **registro de usuários**, **login com sessão**, **dashboard protegido** e **logout**.

---

## Funcionalidades

- Registro de usuário com nome, email e senha
- Login de usuário com validação de senha
- Dashboard acessível apenas para usuários autenticados
- Logout e destruição da sessão
- Feedback visual de erros e sucesso

---

## Tecnologias Utilizadas

- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Backend:** Node.js, Express
- **Banco de dados:** MySQL
- **Segurança:** bcrypt para hash de senhas, express-session para sessões
- **Controle de versões:** Git e GitHub
